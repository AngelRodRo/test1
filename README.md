## 🚀 Getting Started

### Client

```
cd client/
npm install
npm run start
```

### Server

```
cd server/
npm install
npm run start
```

## 🛠 Configuration 

Copy **.env.example** in server folder and use the default params or change for any other values. 

## 🐳 Docker 

You can create a docker image using the **Dockerfile** included on the server folder.
